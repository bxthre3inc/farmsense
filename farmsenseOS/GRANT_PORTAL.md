# Grant Reviewer Portal

## Location
- URL: `/grant` 
- Public: Yes (for grant reviewers to access without login)

## Features

### 1. Overview Tab
- Project summary
- Key stats (19,466 devices, 117,000 acres, $5.38M investment)
- Validation partner: CSU San Luis Valley Research Center
- Key innovations

### 2. Hardware & BOMs Tab
- Complete Bill of Materials
- Device specifications
- Quantity and cost breakdown
- Purpose descriptions

### 3. Timeline Tab
- Phase 1: Pilot Deployment (Q1-Q2 2026)
- Phase 2: Network Expansion (Q2-Q3 2026)
- Phase 3: Integration & Testing (Q3-Q4 2026)
- Phase 4: Full Operation (Q4 2026-Q1 2027)

### 4. Budget Tab
- Total hardware investment
- Annual O&M estimates
- Cost breakdown by device type

### 5. Team Tab
- Principal Investigator
- Technical Lead
- Data Scientist
- Field Operations

### 6. Compliance Tab
- SLV 2026 Groundwater Rules
- State Engineer Requirements
- Water Court Defensibility
- CDWR Integration

### 7. Documents Tab
- Downloadable project documents
- Hardware BOMs
- Technical specifications
- Network architecture
- Compliance framework
