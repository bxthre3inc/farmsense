## 7. HARDWARE NETWORK EVALUATION

### 7.1 Sensor Network Design

**Topology:** Hub-and-spoke (1 hub: 9 fields)

```
Hub Field (1):
├── Jetson Nano (compute)
├── Gateway (LoRa + WiFi + LTE)
└── Master sensor (11 soil nails)

Spoke Fields (8):
└── Master sensor (11 soil nails each)

Total: 9 fields × 11 nails = 99 sensors + 9 masters = 108 total
```

### 7.2 Novelty Assessment

| Element | Standard Approach | FarmSense Approach | Advantage |
|---------|-------------------|-------------------|-----------|
| Per-field compute | 1 unit per field | 1 unit per 9 fields | 89% cost reduction |
| Connectivity | Cellular per field | Shared gateway | 75% cost reduction |
| Power | Grid or battery | Solar + battery | Off-grid capable |
| Failover | None | Triple redundant | 99.9% uptime |

**Hardware Novelty Score: 7/10** - Clever architecture optimizations, not breakthrough technology.

### 7.3 Manufacturability

- **Sensor nails:** Off-the-shelf Decagon/Meter components

- **Installation:** Low-voltage wiring, standard conduit

**Manufacturability Score: 8/10** - Uses standard components, no custom silicon required.
